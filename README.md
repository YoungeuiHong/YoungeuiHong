## 프론트엔드 개발자 홍영의입니다 👋

### 🛠 Tech Stack
* 2년간 React + Spring + MongoDB로 웹 서비스를 개발하는 풀스택 개발자로 근무했습니다.
* 현재는 프론트엔드 개발자로 근무하며, React, Zustand, Styled-Components, Vite를 활용한 개발에 집중하고 있습니다.
* 사이드 프로젝트에서는 Next.js를 주로 사용하며, React Query, PWA 등 다양한 기술을 적용하며 성장하고 있습니다.
